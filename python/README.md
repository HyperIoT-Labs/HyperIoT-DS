# Statistics using PySpark
The statistics algorithms are written in Python and run using pyspark.
The statistics available at the moment are:

 - Mean
 - Standard Deviation
 - Maximum
 - Minimum

Versions of technologies used in the process:

 - [x] Spark 3.4.0.
 - [x] Python 3.8.0.
 - [x] Circe 0.13.0.
 - [x] HBase 2.5.3.
